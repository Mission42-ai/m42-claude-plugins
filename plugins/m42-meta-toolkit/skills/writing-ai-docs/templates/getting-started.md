## Common Issues

### Issue: "Command not found"

**Solution**: Add [Product] to your PATH:

```bash
export PATH=$PATH:/path/to/[product]
```

### Issue: "Permission denied"

**Solution**: Run with appropriate permissions:

```bash
sudo [command]
```

Or fix permissions:

```bash
chmod +x /path/to/[product]
```
