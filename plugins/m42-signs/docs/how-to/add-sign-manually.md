# How to Add a Sign Manually

Add signs directly from your development experience.

---

## Quick Start

```bash
/m42-signs:add
```

Follow the interactive prompts to provide:
- Title
- Problem description
- Solution
- Target CLAUDE.md path

---

## Direct Mode

Skip prompts with `--direct`:

```bash
/m42-signs:add --direct "Title" "Problem description" "Solution" "./CLAUDE.md"
```

---

## Tips

- Use clear, actionable titles
- Be specific about the problem context
- Include concrete steps in the solution
- Target the most relevant CLAUDE.md file
